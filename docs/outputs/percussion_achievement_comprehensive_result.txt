MEASUREMENT CRITERIA SPECIFICATIONS
====================================================================================

PRECISION METRICS:
- Timing Deviation: ±10ms (excellent) to ±100ms (needs work)
- Accuracy Rate: 99%+ (exceptional), 98-99% (excellent), 95-98% (good), 90-95% (acceptable)
- Consistency: <5% variance (perfect), 5-10% (excellent), 10-20% (good), >20% (needs work)
- Dynamic Range: ±3dB (excellent) for volume control
- Spectral Purity: Harmonic clarity and tone consistency

ENSEMBLE METRICS:
- Section Synchronization: All members within ±30ms of each other
- Tonal Blend: <2dB variance in sound pressure across section
- Hand Independence (Tenors): <10ms variance between four drums
- Pocket Lock (Bass/Tenor): ±20ms timing synchronization

ARTISTRY METRICS:
- Dynamic Variation: 8+ distinct dynamic levels across passage
- Tonal Contour: Intentional shape to rolls (attack/sustain/release)
- Musical Interpretation: Deviation from written part judged as improvement

RARITY TIER SYSTEM:
┌──────────┬─────────────┬──────────┬──────────────┬────────────┐
│ Tier     │ Base Points │ Frequency│ Frequency %  │ Multiplier │
├──────────┼─────────────┼──────────┼──────────────┼────────────┤
│ Common   │ 1-3 pts     │ Regular  │ 50-70%       │ 1.0x       │
│ Uncommon │ 4-8 pts     │ Often    │ 20-40%       │ 1.5x       │
│ Rare     │ 9-15 pts    │ Seldom   │ 5-20%        │ 2.5x       │
│ Legendary│ 16-25 pts   │ Rare     │ <5%          │ 4.0x       │
└──────────┴─────────────┴──────────┴──────────────┴────────────┘

POINT CALCULATION:
  Final Points = Base Points × Rarity Multiplier × Difficulty Coefficient
  Where Difficulty ranges 0.5 (easier) to 2.0 (harder)

SEASON CAPS (prevent oversaturation):
  - Common: Unlimited (encourages consistent engagement)
  - Uncommon: 40 points max per season (~5-10 achievements)
  - Rare: 60 points max per season (~4-6 achievements)
  - Legendary: 25 points max per season (1-2 max per performer)

====================================================================================
TRUST SCORE INTEGRATION
====================================================================================

ACHIEVEMENT-TO-TRUST FORMULA:
  trust_delta = points_awarded × 0.5 × rarity_weight[tier]

  Where rarity_weight = {
    'common': 0.05,      # Small trust impact
    'uncommon': 0.15,    # Moderate impact
    'rare': 0.35,        # Significant impact
    'legendary': 0.75    # Major impact
  }

  new_trust = min(old_trust + trust_delta, 100.0)

EXAMPLE CALCULATIONS:
  - Common achievement (2 pts): trust_delta = 2 × 0.5 × 0.05 = 0.05
  - Uncommon achievement (6 pts): trust_delta = 6 × 0.5 × 0.15 = 0.45
  - Rare achievement (12 pts): trust_delta = 12 × 0.5 × 0.35 = 2.1
  - Legendary achievement (20 pts): trust_delta = 20 × 0.5 × 0.75 = 7.5

TRUST SCORE MULTIPLIER (achievers):
  - 20+ achievements in season: 1.3x trust resilience (30% more stable)
  - 10-19 achievements: 1.15x trust resilience
  - <10 achievements: 1.0x baseline

SEASON DECAY EXEMPTION:
  - Each legendary achievement: 2% reduction in decay rate
  - Adjusted decay = max(base_decay - exemption, 0.01%)
  - Prevents new seasons from heavily penalizing legendary performers

DATABASE INTEGRATION:
  achievements_earned.points_awarded = calculated final points
  achievements_earned.rarity_multiplier = tier multiplier applied

ACHIEVEMENT DETECTION ENGINE:
```python
class AchievementDetectionEngine:
    def detect_post_performance(self, performance_data):
        achievements = []

        for performer in performance_data.performers:
            metrics = performance_data.get_metrics(performer.id)
            achievements.extend(self._check_precision(performer, metrics))
            achievements.extend(self._check_ensemble(performer, metrics))
            achievements.extend(self._check_artistry(performer, metrics))
            achievements.extend(self._check_role_specific(performer, metrics))

        for section in performance_data.sections:
            achievements.extend(self._check_section(section))

        return achievements
```
