## Show Concept
Demonstrate the model strategy system end-to-end: corps can evolve strategies, compare model specs, and track performance over time.

## Segments
- Frontend UI build (React components for strategy panels and leaderboards)
- Backend API implementation (FastAPI endpoints for model specs and strategies)
- Test suite creation (pytest coverage for strategy evolution logic)

## Deliverables
- Strategy panel and model leaderboard components
- Model spec and strategy API endpoints
- Unit and integration tests for strategy evolution

## Constraints
- Use v1.ts for API calls
- Follow existing model patterns in backend/models/
- Reuse shared UI primitives (Card, Badge, DataTable)
