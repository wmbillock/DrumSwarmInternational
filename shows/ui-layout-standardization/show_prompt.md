## Show Concept
Establish consistent page layout patterns — containers, headers, back navigation, and summary bars across all frontend pages.

## Deliverables
- Every page uses `page-content` wrapper class with standard max-width/padding via CSS
- Every page gets a `page-header` with title, optional back button, optional status badge
- All detail pages have back navigation to parent list
- Pages with aggregate stats use the summary-bar pattern

## Constraints
- CommandCenter is the baseline reference for layout
- No inline styles for layout concerns — use CSS classes
- Preserve existing page functionality while standardizing structure
