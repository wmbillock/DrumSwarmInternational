**[user]** The show concept is "Compile Time" — a meditation on the tension between creation and execution. Movement 1 "Source Code" opens with brass writing melodic fragments that build into a theme, like code being written line by line. Movement 2 "Build Errors" introduces controlled chaos — deliberate dissonance, broken formations that reassemble, representing debugging. Movement 3 "Deployment" resolves everything into a triumphant full-ensemble statement. The guard uses flags that transition from red (errors) to green (success). Drill formations should evoke circuit board patterns and data flow diagrams.
